This template marks all works (except sound recording works) released under PD-US-expired. For sound recordings, please consider using {{PD-US-record-expired}}.

Usage

{{PD-US-expired |1= |country= |hide_us_warning= |deathyear= |reason= }}

Template parameters

ParameterDescriptionDefaultStatus
1licenseIf the work was not first published in the United States, the name of the license template that applies to the work in its country of first publication.emptyoptional
countrycountryHome or source country (country of origin) of the work. Two-letter country code according to ISO 3166-1. US will hide the non-US work warning.emptyoptional
hide_us_warninghide US warningAny value will cause the warning about non-US works to be hidden. Only use this in templates that include a US license as well. Please use 1 (license) or country instead whenever possible.emptyoptional
deathyeardeath yearYear of author's death. Only used to pass on to the license if it needs it.emptyoptional
reasonreasonReason the image was in Public domain in the source country at the URAA date (January 1, 1996 for most countries). Only used to pass on to the license if it needs it.emptyoptional

To add your language to the list of languages supported by this template, please copy the code of Template:PD-US-expired/en (or any other language version you prefer to translate) and replace the text strings in there (the form below can be used to create a translation, the English version is preloaded in the edit box). Please change the parameter lang from en (or whatever language you are translating) to the language code of your language.
